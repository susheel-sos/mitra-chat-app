# mitra-chat-app
Writing Script for mitra-chat
A Django-based system for conducting secure online exams.


## Apps
- mysite: Chat AI + OneToOne chat App 
- accounts: User authentication
- questions: Question bank
- exams: Exam scheduling
- candidates: Candidate management
- security: Anti-cheating features
- Auditing: Audit logs for Admin and user
## Features
- Chat with user-to-user messaging, online status, unread counts, dual mode (AI + Human chat(file share + delete))
